# DevDiary Frontend (React, TailwindCSS)

React frontend for the DevDiary blog app, built with Vite and Tailwind CSS.
Fetches and displays blogs from a FastAPI backend.
Supports creating and viewing blog posts via API integration.
Run locally with: `npm install && npm run dev`

Deployed on Vercel, available at https://blog-frontend-react1.vercel.app/

FastAPI backend docs are available at https://fastapi-blog-api-au5y.onrender.com/docs
